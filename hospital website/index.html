WELCOME TO HOSPITAL MANAGEMENT SYSTEM



This is the reference of how the website will look like. I Harsh Gupta has created the website for "HOSPITAL MANAGEMENT SYSTEM" using HTML and css. HMS is mainly focused to provide services at administration level.


Hospital Managemnt System is a system enabling hospitals to manage information and data related to all aspects of healthcare- processes, patients and more in a computarize way. The project will automate the daily operations of hospitals.


The project keeps track of the staff and patient details like registration, book an appoinment with doctor the basic information about the services provided by the hospital. The project will reduce the manual processing of each department.



The user will get seamless experience through this project wheather it is to contack the hospital or to register themselves to book an appoinment while just sitting at home and to avoid the hassle to be in queue and also avoiding any human error while manual processing of the documents.
